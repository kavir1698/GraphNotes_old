Introduction

Researching requires you to know many specific facts about different topics. You will also need to know credible references for those facts. To tackle this problem, the first thing that people do is to refer to encyclopedias, such as Wikipedia, or textbooks to find their answers. If you have a university account with access to paid scientific journals, you may also look into journal papers. This is, however, extremely time consuming. If you are writing a research papers, for every paragraph of the introduction of the discussion, you likely need know and cite multiple facts. Going through books and papers to find each fact is hard enough, you will soon forget most of what you have once looked for if you need them in the future.

A traditional method for gathering all the necessary information has been using paper note cards. In a note card, one writes a fact or information they thinks will be useful, along with a reference for it. They would later organize the note cards and use them for writing. This method has a few drawbacks, namely over time, it will be difficult to keep all the cards organized and it will become more and more difficult to find a specific card. Therefore, people would usually make new cards for each new project.

With the GraphNotes, we fix these problems, and introduce new functionalities for researching more efficiently. In this program, you can record every new interesting fact or information (we call them concepts from now on) you come across. For each concept, you may add as many different descriptions with different references as you like. Additionally, you can add descriptions for how two different concepts are related. As a result, whenever you need a specific information, you can search for descriptions of a concept or how two concepts relate.

How to use it?

Initializing the program for the first time

If it is the first time that you open this program, it you can specify a location and name for the program's database. It will be where your data will be stored.

Adding a single concept and its descriptions

  • Click on the Add data button.
  • Type information in the Concept 1 section
    • To make organizing information easier, each concept has a Subcategory field. Use the Subcategory field to add specific details about a more generation concept. For example, if you want to add information about the speed of light, you may type "Light" in the concept field, and type "Speed of" in the subcategory field.
    • The Synonyms field is for writing other ways that the same concept may be written, so that searching for either of them will return the same retuls.
    • The Study field specifies the type of study of the reference, e.g. it can be an experimental or a theoretical study.
    • To add a reference, right click in the white are of the Reference field and click on the New reference option. You may either type in the reference details, import references from a bibtex file, or most conveniently, if you have a DOI, enter it and the program will grab the reference automatically.
    • You may leave any field blank.

Adding a relation between two concepts

  • Click on the Add data button.
  • Type the first concept, and if you like a Subcategory for the first concept.
  • Type the second concept, and if you like a Subcategory for the second concept.
  • Type a description for the relation between the two concepts in the Relatin field.
  • You may add a study type in the Study field.
  • To add a reference, wright click in the white are of the Reference field and click on the New reference option. You may either type in the reference details or import references from a bibtex file.

Searching

  • Type in a work in the Search field in the main window.
  • Use check boxes below the search field to specify where to search.
  • The search results appear in the Concepts area.

All the steps above can be repeated, i.e. a concept can have multiple descriptions and relations.
